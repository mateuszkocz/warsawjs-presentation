What is Angular 2 all about?

  • Mobile devices
  • Evergreen browsers (and old Android)
  • Slimming down
  • Uses TypeScript and EcmaScript 6

No longer in Angular 2

  • controllers (responsibilities taken by components)
  • $scope and it's methods
  • directives as we know them today
  • jqLite
  • services, modules, multiple ng-* directives, config and run phases

What about services?

Angular will no longer provide any frame for services. Instead, they will be plain old JavaScript [place your type here].

                // A service's example
                export SomeBusinessLogic {
                  addTwoNumbers(number1:number, number2:number):number {
                    return number1 + number2;
                  }
                }

Will not tell about:

  • router,
  • http,
  • change detection,
  • tests.
